========================
OVN tmt CI Experiment
========================

An experimental fork of `OVN <https://github.com/ovn-org/ovn>`_ that evaluates
`tmt <https://tmt.readthedocs.io/>`_-based test plans from
`ovn-tmt-tests <https://github.com/NicholasBHubbard/ovn-tmt-tests>`_ as a
replacement for GitHub Actions CI. The original workflows remain available for
manual runs while the tmt coverage and results are brought to parity.
