========================
OVN tmt CI Experiment
========================

An experimental fork of `OVN <https://github.com/ovn-org/ovn>`_ that replaces
GitHub Actions CI with `tmt <https://tmt.readthedocs.io/>`_-based test plans
from `ovn-tmt-tests <https://github.com/NicholasBHubbard/ovn-tmt-tests>`_.
